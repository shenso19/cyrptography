# Compilation
A Makefile is included. To compile the program, run "make" (bash)
to remove built artifacts, run "make clean'
to run the executable - "make run"
The program will run all three test caes (AES-128, AES-192, AES-256) and print the plaintext, key, encrypted ciphertext, and decrypted result for each.
